#🛒 Blinkit Sales Analytics Dashboard

📌 Project Overview

This Power BI dashboard analyzes Blinkit's grocery sales performance, outlet trends, product categories, and customer purchasing patterns. The dashboard helps identify business opportunities and supports data-driven decision-making.

---

🎯 Objective

* Analyze overall sales performance
* Monitor outlet performance across locations
* Identify top-performing product categories
* Track sales trends and KPIs
* Generate actionable business insights

---

🛠 Tools & Technologies Used

* Power BI
* DAX
* Power Query
* Excel

---

 📈 Key Features

* Total Sales Analysis
* Outlet Performance Analysis
* Product Category Insights
* Sales KPI Dashboard
* Outlet Size & Location Analysis
* Interactive Filters and Slicers
* Revenue Trend Tracking

---

📊 Business Insights

* Identified top-performing outlet types
* Analyzed sales contribution by product category
* Evaluated outlet performance by location
* Tracked key business KPIs
* Discovered sales trends and growth opportunities

---

📂 Dataset

Blinkit Grocery Sales Dataset
